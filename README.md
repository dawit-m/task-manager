# Task Manager CLI (Java)

## Description
A simple command-line Task Manager built using Java.

This project demonstrates Object-Oriented Programming concepts such as classes, constructors, encapsulation, and separation of concerns. It allows users to add, list, complete, and delete tasks.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks

## Tech Stack
- Java
- OOP Concepts
- ArrayList
- Git & GitHub
